Añade la dependencia `ideascloud-cli` en tu dispositivo de forma global:

<code-group>
  <code-block label="Yarn" active>

```bash
yarn global add ideascloud-cli
```

  </code-block>
  <code-block label="NPM">

```bash
npm install -g ideascloud-cli
```

  </code-block>
</code-group>
